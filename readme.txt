============
Requirements:
============

The following software needs to be installed:

1) Graphviz: https://www.graphviz.org/ 
2) The Ruby interpreter: https://www.ruby-lang.org
3) The R programming language: https://www.r-project.org/ 
4) RStudio: https://www.rstudio.com/
5) The Ruby Gem Ruby-Graphviz: https://github.com/glejeune/Ruby-Graphviz

Finally, the Graphviz/bin folder should be added to the path environment variable of your system

========
Examples:
========

Folder "examples" includes (i) a real case study coming from the Pontificia Universidad Catolica de Valparaíso (PUCV), (ii) and a fictional one created with random data. 
To run the examples, go to the folders and execute the "runme.bat" files.
